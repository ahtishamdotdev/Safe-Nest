


# 🏠 Safe Nest  


Safe Nest is a simple **static web page** built using only **HTML, inline CSS, and JavaScript**.  
It demonstrates a minimal design with embedded styles and scripts, making it lightweight and easy to run.  

🌐 **Live Demo (Netlify):** [familysafenest.netlify.app](https://familysafenest.netlify.app/)

## 🚀 Features  
- Single-page application (`index.html`)  
- Uses **inline CSS** for styling  
- Uses **inline JavaScript** for interactivity  
- No external dependencies – runs directly in the browser  

## 🛠️ Tech Stack  
- **HTML5**  
- **CSS3 (inline)**  
- **JavaScript (inline)**  

## 📂 Project Structure  
```

Safe-Nest/
│
├── index.html          # Main file with inline CSS & JS

````

## ⚙️ How to Use  

### 1️⃣ Clone the Repository  
```bash
git clone https://github.com/ahtishamdotdev/Safe-Nest.git
cd Safe-Nest
````

### 2️⃣ Open in Browser

Simply open the `index.html` file in any web browser. 🚀

No setup or dependencies required.
